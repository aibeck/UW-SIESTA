Home Page

This website is Sleep Indentification Enabled by Supervised Training Algorithms (SIESTA) -- a web-based method of algorithmically scoring sleep stages using electrocorticographic (ECoG) and electromyographic (EMG) data from mice.
Get started by extracting features from your data. These features will be run through a previously trained model (or a new model you can train on this website), which will give you sleep scores. If you'd like to include your own data to train a new model, you should first merge the extracted data with your own scores.
This project is completely open-source -- all code is available on our GitHub.
Note: This version is a prototype.
